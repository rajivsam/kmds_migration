# dd_parser_cleaner_migration

This repository contains migration examples for the tool:
https://github.com/rajivsam/dd-parser-cleaner/

## Purpose

The goal is to demonstrate how to apply `dd-parser-cleaner` to real migration projects of different sizes.

## Repository Layout

- `sba_migration/`: largest, end-to-end migration example
- Additional migration example directories can be added at this same level

## How To Use

1. Pick a migration directory.
2. Read that directory's `README.md`.
3. Run the documented workflow for that example.

## Why This Repo Structure

A single parent repository with multiple example directories makes it easy to:

- compare approaches across datasets/projects
- reuse shared documentation patterns
- keep each migration self-contained while still grouped in one showcase
